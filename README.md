Welcome to E-Learn — a powerful B2B e-commerce platform designed to connect sellers with buyers for seamless online learning content transactions. E-Learn empowers educators to retail their courses, and learners to explore, purchase, and access high-quality learning materials. Built with Django, Django REST Framework, Elasticsearch, and React, E-Learn provides a robust, scalable, and secure environment to manage courses, user authentication, and more.

Features

Custom User Authentication: Secure signup, login, and password reset system using JWT tokens.
Course Management: Easy course creation and management for educators.
Image Upload & Handling: Integrated with Pillow for smooth media handling.
Elasticsearch Integration: Efficient course search functionality for users.
Token-Based Authentication: Ensuring secure access to resources.
CORS Support: Seamless communication between the Django backend and React frontend.


Tech Stack

Backend: Django, Django REST Framework, Elasticsearch
Frontend: React, Tailwind CSS
Database: SQLite (default, can be switched to PostgreSQL)
Media Handling: Pillow library for handling images
Authentication: JWT (JSON Web Token)


Prerequisites

Ensure you have the following installed on your system:
Python 3.10+
Node.js & npm (for the frontend)
Elasticsearch (running on localhost:9200)
